# Security Policy

## Supported Version

The newest release is the supported one.

## Reporting A Security Problem

Please do not open a public issue for a real security problem in this repo.

Instead:

1. Use GitHub private vulnerability reporting if it is turned on
2. If it is not turned on, contact the repo owner privately
3. Share steps to reproduce the issue and why it matters

## Important Note

This repo includes intentionally vulnerable training software such as Juice Shop.

If the problem is in that upstream software itself, report it to the upstream project unless this repo caused the issue through its own setup or config.
