# Code Of Conduct

Be respectful.

This project is for learners and contributors with different skill levels. We want people to feel safe asking basic questions.

## Good Behavior

- Be kind
- Be clear
- Help people learn
- Give useful feedback without being rude

## Not Okay

- Harassment
- Discrimination
- Personal attacks
- Sharing someone else's private info
- Trying to make the space hostile

## If There Is A Problem

The maintainers may remove comments, close threads, or block people if needed.

If something goes wrong, contact the repo owner privately.
